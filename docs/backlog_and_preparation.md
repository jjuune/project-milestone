## 🗂️ 1. 프로젝트 기능 백로그 (Backlog)

총 4개의 대형 에픽(Epic)으로 나누어, 향후 개발할 기능 목록을 정리했어.

### 🧱 Epic 1: DB & 대시보드 기초 빌드
* [x] Supabase 기본 DDL 스키마 설계 (`vehicles`, `mileage_logs`, `maintenance_logs`)
* [x] Vite + React + Vanilla CSS 프리미엄 다크 테마 대시보드 레이아웃 구축
* [x] 환경변수 미세팅 시 작동할 `localStorage` 기반 로컬 데모(Mock) 모드 개발
* [x] 어드민 핀코드 인증 및 로 데이터 수정/삭제 권한 통제 기능
* [x] 기아 커넥트 모바일 앱 감성 프리미엄 와이드 히어로 스크린 UI 튜닝

### ⚙️ Epic 2: 차량 등록 및 소모품 고도화 (Hold ⏸️)
* [x] 신규 차량 등록 시 최근 소모품 교체 이력(누적거리, 교체일) 동시 입력 UI/UX
* [ ] **파워트레인별(가솔린, 디젤, 전기차) 소모품 가변 렌더링**
  * 디젤 차량: 요소수(AdBlue) 보충 주기 게이지, DPF 클리닝 알림 위젯 활성화
  * 전기차(EV): 엔진 관련 소모품 가이드 제거, [감속기 오일, 타이어 로테이션, 저전도 냉각수] 위젯으로 자동 스위칭
* [ ] **행정 정보 알림**: 자동차 정기검사일, 법인 자동차보험 갱신일 기반 D-Day 알림 배치

### 🤖 Epic 3: QR 코드 기반 텔레그램 봇 연동 (Hold ⏸️)
* [ ] **차량용 QR 코드 생성 및 텔레그램 딥 링킹(Deep Linking) 파이프라인**
  * 차량 내부에 배치할 QR에 봇 다이렉트 딥링크(`t.me/봇?start=car_id`)를 연결하여 차량 고유 식별
* [ ] **n8n 메시지 라우팅 & ForceReply 처리**
  * QR 스캔 시 텔레그램에서 자동으로 차량을 인지하고 해당 차량의 누적 주행거리 답장을 유도
  * 수신된 주행거리 숫자의 유효성 검사 (이전 기록 대비 역전 확인 등) 및 Supabase DB 저장 자동화

### 📊 Epic 4: 법인 차량 운행일지 및 자동 세무 보정 (Hold ⏸️)
* [ ] **운행일지 DB 스키마 구축** (`driving_logs` 생성)
* [ ] **주 1회 입력 기반 국세청 일지 자동 쪼개기(Generate) 엔진**
  * 직원이 금요일 퇴근 전에 최종 주행거리만 입력하면, 총무팀이 설정해 둔 차량별 기본 주행 패턴(예: 출퇴근 기본 왕복 + 외근 요일 배분)에 맞게 일별 운행 내역을 합법적인 정합성에 맞춰 시스템 내부에서 자동 가상 생성
* [ ] **대시보드 내 세무 지원 모듈**
  * 월별/차량별 실시간 운행 기록 조회
  * 국세청 표준 양식에 맞춘 운행기록부 엑셀(CSV) 1초 다운로드 기능

## 🛠️ 2. 오빠가 준비해야 할 사전 준비 체크리스트 (Preparation Checklist)

개발을 시작하고 실제 라이브 서버로 구동하기 위해 오빠가 미리 세팅하거나 넘겨주어야 할 정보들이야.

| 준비 항목 | 세부 내용 | 용도 |
| :--- | :--- | :--- |
| **① 텔레그램 봇 토큰** | 텔레그램 `@BotFather`를 통해 새 봇 개설 후 `HTTP API Bot Token` 복사해두기 | n8n과 텔레그램 메신저 간 통신 트리거 용도 |
| **② Supabase 개설** | [supabase.com](https://supabase.com)에 로그인 후 무료 프로젝트 1개 생성 및 `URL`, `Anon Key` 복사해두기 | 실시간 주행 및 정비 데이터 DB 저장소 |
| **③ 차량 이미지 (PNG)** | 신차 및 기존 차량의 **배경이 투명하게 처리된 PNG 이미지** 준비 (가로 해상도 600px 이상 권장) | 대시보드 히어로 섹션 메인 비주얼 장식 |
| **④ 차량 상세 정보** | 각 차량의 **모델명, 정식 출고일/인수일, 인수 당시 누적 주행거리(km)** 정보 | 대시보드 D-day 및 소모품 수명 계산 기준점 |
| **⑤ 회사 기본 운행 패턴** | 예: 출퇴근 왕복 기본 주행거리(km), 주요 외근지 왕복 거리 등 | 국세청 제출용 운행일지 가상 자동 쪼개기 알고리즘 기준값 세팅 |
| **⑥ 대시보드 패스워드** | 관리자 권한을 획득할 때 입력할 비밀번호 (기본값 `admin1234` 변경용) | 대시보드 편집/삭제 권한 보안용 |
