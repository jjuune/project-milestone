# PITSTOP (코드네임: Milestone)
> **"아 뭐였더라?"를 없애는 차량 관리 경험**
> 
> *Remember Less. Maintain Better.*

---

## 📌 1. 프로젝트 개요 (Overview)
**PITSTOP (개발 코드명: Milestone)**은 소규모 사업장(1~5대 차량)의 담당자가 법인 차량에 대한 세부 정보를 일일이 기억할 필요 없도록 지원하는 **차량 정보 및 이벤트 타임라인 기억 저장소**입니다.

이 프로젝트는 복잡한 ERP나 배차 운영 툴이 아닙니다. 차량과 관련된 사소하지만 중요한 정보들(정비 시점, 보험사 정보, 검사 일정 등)을 한곳에 안전하게 보관하고, 필요한 시점에 쉽게 꺼내볼 수 있도록 단순함과 직관성에 초점을 맞춥니다.

## 🎯 2. 제품 철학 (Product Principles)

### 🚫 하지 않는 것 (Don't)
* **배차 및 운전자 관리**: 복잡한 사내 스케줄 조율 및 권한 매칭
* **비용 분석 및 회계 ERP**: 예산 관리 및 감가상각 비용 처리 시스템
* **운행일지 자동 생성 (보류)**: 법적 리스크 방지를 위한 가상 운행 쪼개기 기능
* **과도하게 복잡한 정비 관리**: 대기업 수준의 FMS 관제 시스템

### 🟢 하는 것 (Do)
* **차량 상태 요약**: 오늘 당장 해야 할 조치 사항 직관적 노출
* **차량 정보 보관 (Wallet)**: 기억하기 어려운 차량 번호, 보험 계약, 규격 정보 영구 보관
* **이벤트 기반 히스토리**: 정비 내역을 넘어 차량에 일어난 모든 이벤트를 타임라인으로 기록
* **스마트 알림 제공**: 주행거리 및 날짜 기준 조치 필요 사항 선제적 안내
* **긴급 조치 지원**: 비상시 원클릭으로 보험사 긴급출동 연결

## 🧩 3. 핵심 컨셉 (Core Concepts)

### ① 대시보드 (Dashboard)
* **목적**: "오늘 내가 무엇을 조치해야 하는가"에 대한 직관적 피드백 제공.
* **표시 정보**:
  * 🚨 조치 필요 개수 요약
  * 🔴 정기검사 D-12, 🟡 엔진오일 850km 남음 등 액션 아이템 리스트
  * 전체 차량 대수 및 차량별 상태 요약 (정상 / 확인 필요 / 조치 필요)

### ② 차량 카드 (Vehicle Card)
* **UX**: 모바일 및 웹앱에서 슬라이드/스와이프 제어를 통해 여러 차량을 쉽게 전환.
* **상태 시각화**: 🟢 정상, 🟡 확인 필요, 🔴 조치 필요 3단계 상태 및 소모품 수명 게이지 바 노출.

### ③ 차량 정보 지갑 (Vehicle Wallet)
* **목적**: 차량 상세 제원을 기억해야 하는 고통을 제거.
* **보관 데이터**: 차량번호, 차종, 연료 유형, 보험사 및 보험 만기일, 긴급출동 번호, 정기검사 예정일, 수동 메모.
* **확장 예정**: 타이어 규격, 배터리 규격, 등록증 및 보험증권 이미지.

### ④ 이벤트 기반 타임라인 (Event Timeline)
* **목적**: 차량에 일어난 모든 중요 사건(정비, 갱신 등)을 시간 순서대로 보관하는 차량 히스토리북.
* **10초 퀵 입력 UX**: `[+] 버튼` ➔ `이벤트 항목 선택(엔진오일, 타이어 등)` ➔ `현재 주행거리 입력` ➔ `영수증 사진 첨부` ➔ `저장`.

### ⑤ 긴급출동 (Emergency Assistance)
* 대시보드 상에서 `📞 긴급출동` 버튼을 누르면 해당 차량이 가입한 보험사의 긴급출동 번호로 즉시 전화 연결을 지원하는 핫링크 기능.

## 📡 4. 진행 중인 검증 및 개선 방향 (NFC & UX)

* **NFC 기반 딥링크 스캔 트리거**:
  * 차량 내부(콘솔 트레이 등)에 부착된 NFC 카드를 스캔하여 텔레그램 봇 딥링크(`https://t.me/{BotName}?start={vehicle_id}`)를 작동시키는 오프라인 연동 실증.
  * NTAG215 락 상태 검증에 따라 직접 쓰기 모드 또는 고유 UID 매핑 모드 결정.
* **주행거리 오입력 방지 컨펌 단계 추가**:
  * 텔레그램 수동 입력 시 오타로 인한 DB 꼬임을 예방하기 위해, 입력 후 **"직전 주행거리 대비 증가량(XX km)"**을 한번 더 보여주고 승인하는 안전장치 적용 예정.

## 📅 5. 개발 로드맵 (Milestones)

### 🏁 MVP (V1)
* **차량/이벤트**: 차량 및 이벤트 정보 등록/수정/삭제 CRUD
* **대시보드**: 차량 상태 요약, 할 일 목록, 현재 주행거리 입력
* **타임라인**: 이벤트 시간순 정렬 조회
* **알림**: 보험 만기일, 정기검사 예정일, 소모품(엔진오일, 타이어, 브레이크) 알림
* **Wallet & Emergency**: 차량 기본 스펙 저장 및 보험사 긴급출동 핫다이얼 연결

### 🔄 V1.5
* **파일 첨부**: 정비 영수증 및 내역 사진 첨부 기능
* **상세 메모**: 각 타임라인 이벤트별 텍스트 메모 기능 추가

### 🧠 V2 (AI Assistant)
* **OCR 영수증 분석**: 정비 영수증 사진을 업로드하면 항목과 금액을 AI가 자동 인식
* **AI 입력 추천**: 인식된 텍스트 기반으로 알맞은 이벤트 종류(예: 타이어 교체)를 추천 제안
* **문서 보관소**: 차량 등록증 및 보험 증권 고화질 파일 영구 보관
